# DevOps Maven Web Application

This project demonstrates how to build and containerize a Java web application using Maven and Docker as part of a DevOps workflow.

The application is packaged as a WAR file using Apache Maven and deployed inside an Apache Tomcat container using Docker.

## 🚀 Project Overview

This project simulates a real DevOps workflow where a Java application is:

1. Built using Maven
2. Packaged as a WAR artifact
3. Containerized using Docker
4. Deployed using Apache Tomcat

## ⚙️ Technologies Used

- Apache Maven – Build and dependency management
- Docker – Containerization
- Apache Tomcat – Web application server
- Git & GitHub – Version control
